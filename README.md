# PortCity

About the Project

The PortCity Project is an interactive website that showcases a marina and seaside activities. Users can explore information about the marina, view events, and get in touch through the contact page.

The project is developed using HTML and CSS and includes all essential web design structures.

Pages Included

index.html – Home Page

about.html – About PortCity

services.html – Activities & Locations

contact.html – Contact Page

Additional pages can be added if desired.

HTML Elements Used

The project includes the following HTML elements and structures:

1. Text and Headings

Headings from <h1> to <h6>

Paragraphs

Bold, italic, and underlined text

2. Styles and Colors

Colors and text styles applied using CSS

3. Block Elements and Layout

Divisions and spans for page layout

Top menu, footer, and sectioned page design

4. Images

At least two images included on the pages

At least one image is used as a clickable link

5. Links

Navigation links between pages

External links such as Google Maps, Wikipedia, social media, and municipal websites

6. Lists

At least one ordered list

At least one unordered list

7. Table

A table displaying a weekly activity schedule, including headers and rows

8. Iframe

At least one iframe included

Examples: Google Maps, Wikipedia pages, or embedding another HTML page

Folder Structure Example

index.html

about.html

services.html

contact.html

images/ folder for pictures

Usage

Download the project files to your computer.

Open index.html in your browser.

Use the menu to navigate to other pages.

Images, tables, lists, and iframe content will display on the pages.
